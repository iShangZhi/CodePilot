# Plugin: file-sync

通用文件目录同步能力（先删后复制，确保目标与源完全一致）。

## 能力描述

将源目录完整同步至目标目录，采用先删后复制策略，消除目标目录残留文件。

## 执行步骤

```bash
# 1. 删除目标目录（含所有子文件）
rm -rf "{dest}"

# 2. 复制源目录至目标位置
cp -r "{src}" "{dest}"

# 3. 验证：对比文件列表
ls "{src}/"
ls "{dest}/"
```

## 验证规则

- 两侧文件列表完全一致 → 同步成功
- 目标目录文件数 ≠ 源目录 → 报告差异，终止后续操作
- 源目录不存在 → 报错终止，不执行删除

## 输出格式

```markdown
### 同步结果
- 源：{src}
- 目标：{dest}
- 文件数：{N}
- 状态：✓ 成功 / ✗ 失败（原因：...）
```

## 注意事项

- 执行前必须确认源目录存在，避免误删目标
- 不支持增量同步，始终全量替换
- 软链接作为链接本身复制，不追踪链接目标

## 使用方 Agent 调用约定

```
调用 file-sync，src：{源路径}，dest：{目标路径}
同步完成后输出验证结果
```
